How to Traslating documentation

모든 번역과정은 Transifex 에서 이루어진다.

1. Django i18n mailing list 에 접속해 자신에 대해 소개한다.
2. Transifex 가입 후, Django project page에 방문한다.
3. Django project page에서 작업 할 언어를 선택한다. 또는 언어가 아직없는 경우에는 "언어 요청"링크를 클릭하고 적절한 언어를 선택하여 새 언어 팀을 요청한다.
4. 그 후에 "join this team" 버튼을 클릭하면 이 팀의 회원이 된다. 모든 팀에는 회원 요청을 검토 할 담당자가 적어도 한 명 있고, 물론 팀 코디네이터에게 연락하여 절차 상 문제를 분명히하고 실제 번역 프로세스를 처리 할 수도 있다.
5. 팀원이 되면 팀 페이지에서 업데이트 할 번역 리소스를 선택한다. 예를 들어 'core' 리소스는 모든 비 기여 번역을 포함하는 번역 카탈로그를 나타낸다. 각 contrib 앱에는 리소스도 있다.
6. 번역에 참여하면 project 내 translator로 인정을 받고 계속해서 활동을 할 수 있다.
