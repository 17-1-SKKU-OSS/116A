Tutorial 1. Django install

Install pip program
  • pip 프로그램은 파이썬의 오픈소스 저장소인 PyPI에 있는 SW 패키지를 설치하고 관리해주는 명령입니다. 그래서 파이썬에서 많이 사용되는 패키지들은 pip 명령으로 간단히 설치가 가능합니다. 장고 프로그램 역시 pip 프로그램으로 설치를 진행하는 것이 가장 간단합니다. 만일 여러분의 시스템에 pip 프로그램이 설치되어 있지 않다면 아래 명령들을 순서대로 입력하여 pip 프로그램을 먼저 설치합니다. 아래 디렉토리 또는 자신이 원하는 임의의 디렉토리에 get-pip.py 파일을 가져오면 됩니다.

Step 1. #cd /usr/local/src

아래 주소에 있는 get-pip.py 파일을 가져오면 되는데, 다음 명령처럼 wget을 사용해도 되고, 또는 브라우저로https://pip.pypa.io/en/latest/installing.html#install-pip 사이트에 접속하여 해당 링크를 클릭해서 다운받아도 됩니다.

Step 2. # wget https://bootstrap.pypa.io/get-pip.py

다음 명령을 실행하면 /usr/bin/pip처럼 시스템 bin 디렉토리에 pip 프로그램이 설치됩니다.

Step 3. # python get-pip.py

pip 프로그램이 준비되었으면, step 4 명령을 입력하여 간단하게 Django를 설치할 수 있습니다.

Step 4. # sudo pip install Django

Django 설치가 완료됩니다.
